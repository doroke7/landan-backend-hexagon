## жңҚеӢҷжӢ“жЁё

- **facade**пјҡе°ҚеӨ– gRPC е…ҘеҸЈпјҢ`register` / `table` е…©еҖӢ handler зӣ®еүҚжҳҜ stubпјҢйӮ„жІ’жҺҘ usecaseгҖӮ
- **resource**пјҡе…§йғЁиіҮж–ҷжңҚеӢҷпјҢзӣҙжҺҘи®ҖеҜ« mysqlпјҢеғ…дҫӣ facade / http е‘јеҸ«пјҲ`AdminUserUsecase` е°Ҳеұ¬ж–јйҖҷжўқи·Ҝеҫ‘пјҢиө° `usecase/application/any/model`пјүгҖӮ
- **http**пјҡGin REST APIпјҢзӣ®еүҚеҸӘжңү `Admin/Authentication/Authenticator/SignIn` йҖҷеҖӢзҷ»е…Ҙз«Ҝй»һпјҢиө°е°Ҳеұ¬зҡ„ `AuthenticatorUsecase`пјҲ`usecase/application/any/admin/authentication`пјүпјҢйҖҸйҒҺ gRPC е‘јеҸ« resource жңҚеӢҷжҹҘеёіиҷҹгҖӮ
- **cron / consumer / command**пјҡдёүеҖӢйҖұйӮҠијёе…ҘдҫҶжәҗпјҢеҗ„иҮӘи§ёзҷјгҖҢе№« AppUser еҠ йӨҳйЎҚгҖҚпјҢе…ұз”ЁеҗҢдёҖд»Ҫ `AppUserUsecase.IncreaseBalance`пјҲ`usecase/application/any/admin/resource`пјүпјҢеә•еұӨйғҪжҳҜзӣҙжҺҘжү“ mysqlгҖӮдёүиҖ…е·®з•°еҸӘеңЁеҸғж•ёжҖҺйәјдҫҶпјҡcron жҺ’зЁӢеҜ«жӯ»гҖҒconsumer и§Ј MQ payloadгҖҒcommand еҗғ CLI flagгҖӮ
- **websocket / client**пјҡе®№еҷЁйғҪйӮ„еңЁпјҢдҪҶзӣ®еүҚжІ’жңүжҺӣд»»дҪ• handlerпјҢжҢҮд»Өе•ҹеӢ•еҫҢдёҚжңғиҷ•зҗҶд»»дҪ•и«ӢжұӮпјҲжҳҜд№ӢеүҚжӢҝжҺүиҲҠ `User` еһӮзӣҙеҲҮйқўеҫҢз•ҷдёӢзҡ„з©әж®јпјҢзӯүжңүж–°йңҖжұӮеҶҚиЈңпјүгҖӮ

дҫқиіҙж–№еҗ‘ж°ёйҒ жҳҜгҖҢеӨ–еұӨжҢҮеҗ‘е…§еұӨгҖҚпјҡ`input adapter вҶ’ usecase/port вҶ’ usecase вҶ’ output/port вҶҗ output adapter`пјҢ
`usecase` е®Ңе…ЁдёҚзҹҘйҒ“иҮӘе·ұиў« http йӮ„жҳҜ grpc йӮ„жҳҜ cron е‘јеҸ«пјҢд№ҹдёҚзҹҘйҒ“иіҮж–ҷеҲ°еә•еӯҳеңЁ mysql йӮ„жҳҜиө° gRPC иҪүзҷјгҖӮ
